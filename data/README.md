# Data Folder

This folder contains **sample data only**.

The full ISCX IDS 2012 dataset is not included due to size and licensing restrictions.

## Structure
- `raw/` — small sample of raw network flows  
- `processed/` — small sample of cleaned and encoded data  

## How to Download Full Dataset
You can obtain the full dataset from the official ISCX repository:
https://www.unb.ca/cic/datasets/ids.html

## Important
Do NOT upload full datasets to GitHub.
